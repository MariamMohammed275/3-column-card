# 3-Column Preview Card Component

## Overview

This project is a 3-Column Preview Card built with HTML and CSS. 
The design consists of three cards representing different vehicle categories: Sedans, SUVs, and Luxury vehicles.
Each card contains an icon, heading, descriptive text, and a "Learn More" button.

The project was created to practice CSS layout techniques, spacing, positioning, typography, and Flexbox.

## Features

- Three-column card layout
- Custom styling for each card
- Vehicle icons and themed color schemes
- Styled call-to-action buttons
- Flexbox-based layout structure
- Clean and organized code structure

## Technologies Used

- HTML5
- CSS3
- Flexbox

## What I Learned

During this project, I practiced:

- Structuring web pages with semantic HTML
- Creating layouts using Flexbox
- Applying custom colors and typography
- Using relative and absolute positioning
- Managing spacing with margin and padding
- Styling buttons and card components
- Organizing CSS for readability and maintenance
